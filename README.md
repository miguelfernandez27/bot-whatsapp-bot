# WhatsApp Bot BARBERIA

Sistema automatizado de gestión de turnos para barbería integrado con WhatsApp Cloud API, Google Calendar y Google Sheets utilizando Google Apps Script.


## Descripción

Este proyecto fue desarrollado con el objetivo de automatizar la reserva y administración de turnos mediante WhatsApp, centralizando la disponibilidad en Google Calendar y registrando todas las operaciones en Google Sheets.

El sistema permite a los clientes:
- reservar turnos
- consultar disponibilidad
- cancelar reservas
- interactuar mediante botones y listas dinámicas de WhatsApp

Además, incorpora mecanismos de:
- control de concurrencia
- prevención de spam
- persistencia de estados
- validación de disponibilidad en tiempo real

# Tecnologías Utilizadas

- JavaScript (Google Apps Script)
- WhatsApp Cloud API
- Google Calendar API
- Google Sheets API
- Meta Graph API
- Git
- GitHub
- CLASP
